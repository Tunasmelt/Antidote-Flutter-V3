# Antidote Flutter

This is the Flutter mobile application for Antidote v3 - Music Playlist Analytics Platform.

## Project Structure

- `antidote_flutter/` - Main Flutter project directory
  - `lib/` - Dart source code
  - `assets/` - Images, fonts, and animations
  - `test/` - Unit and integration tests
  - `pubspec.yaml` - Flutter dependencies

## Getting Started

1. Navigate to the Flutter project:
   ```bash
   cd antidote_flutter
   ```

2. Install dependencies:
   ```bash
   flutter pub get
   ```

3. Run the app:
   ```bash
   flutter run
   ```

## Requirements

- Flutter SDK >= 3.16.0
- Dart SDK >= 3.2.0
